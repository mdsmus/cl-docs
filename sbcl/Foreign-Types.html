7.2 Foreign Types

Alien types have a description language based on nested list structure. For example the C type 

     struct foo {
         int a;
         struct foo *b[100];
     };

has the corresponding SBCL FFI type 

     (struct foo
       (a int)
       (b (array (* (struct foo)) 100)))
