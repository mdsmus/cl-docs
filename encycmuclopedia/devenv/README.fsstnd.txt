The binary is in /usr/bin/lisp.
The images and auxiliary files are in  /usr/lib/cmucl.
The documentation in /usr/doc/cmucl.
The generated images by config are placed in the home directory.
The site-init.lisp file is in /var/lib/cmucl (non-shareable because
it contains the site-name!)
The info-files in /usr/info/
The source in /usr/src/cmucl

If there are problems, or errors in this arrangement,
